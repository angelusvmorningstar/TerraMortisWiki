## What this page is for

The core books are dense, and the chronicle changes some of their calls. This
page will explain those house rules in ordinary language, so a player can find an
answer without hunting through a PDF or re-asking the Storyteller. The real
entries are pending.

## How a finished entry will read

Each entry will name the rule, state the change plainly, and give a short worked
example. As a shape only:

- **Rote actions.** House rule: *roll twice, keep the better result*.
- **Feeding pools.** Declared and validated, then rolled live, not invented.
- **Merit actions.** Judged on their content, not rolled unless a rule says so.

An ordered worked example, for layout purposes only:

1. Read the rule as written in the core book.
2. Read the house change here in plain words.
3. Apply the change; if in doubt, ask the Storyteller.

A closing reminder:

> When a house call and a book call disagree, the house call wins at this table.

Inline, a term like a `Resolve + Composure` reflexive still reads as code, and a
worked note still sits in a block:

```
Book:   rote grants 8-again
House:  rote = roll the pool twice, keep the better result
```
